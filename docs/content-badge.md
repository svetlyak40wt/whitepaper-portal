# Бейдж

![Бейдж](_images/badge.png)

Блок для маркировки статуса сущности. Например, статус платежа.

```js
{
	block: 'badge',
	mods: { view: 'default' },
	content: 'Badge'
}
```

| Модификатор | Значение                            |
|-------------|-------------------------------------|
| view        | default / success / warning / alert |

## Задание

Расставь `badge` соответственно статусам в строках платежей.
